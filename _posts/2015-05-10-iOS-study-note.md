================ Objective C (pros and cons)  ================<br>
- Pros
- Runtime (mini and powerful)
- Base on C (current compile compatible and lots of library available)
- Cons
- Garbage collection
- Namespace
- Operator override
- Multiple inheritance
- Dynamic type make it hard for compiler optimization
